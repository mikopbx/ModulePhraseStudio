# ModulePhraseStudio — Студия фраз

Серверная генерация голосовых фраз (TTS) для MikoPBX на базе [Piper TTS](https://github.com/rhasspy/piper).

## Возможности

- Генерация `.wav` файлов из текста прямо на сервере MikoPBX
- 25+ языков через загружаемые голосовые модели Piper
- UI в админке: вводите фразу, выбираете голос, жмёте «Сгенерировать», слушаете, скачиваете
- Пустая установка — бинарь Piper и модели голосов загружаются по требованию из UI
- Кэширование сгенерированных фраз по `md5(текст + голос + sample_rate)` — повторная генерация не нужна
- Опциональный ресэмплинг в 8 кГц моно для прямого использования в телефонии

## Архитектура

```
Пакет модуля            — компактный (~100 КБ), без бинарей
Бинарь движка           — загружается в db/piper/ по требованию
Голосовые модели (.onnx) — загружаются в db/voices/ по требованию (~30–60 МБ каждая)
Сгенерированные фразы    — кэшируются в db/phrases/
```

## REST API (v3, авто-обнаружение)

Базовый путь: `/pbxcore/api/v3/module-phrase-studio`

| Метод  | Эндпоинт                       | Описание                              |
|--------|--------------------------------|---------------------------------------|
| GET    | `engine`                       | Статус движка (установлен, версия)    |
| POST   | `engine:install`               | Скачать и установить бинарь Piper     |
| DELETE | `engine`                       | Удалить бинарь Piper                  |
| GET    | `voices`                       | Каталог + установленные голоса        |
| POST   | `voices:install`               | Скачать модель голоса по ID           |
| DELETE | `voices/{id}`                  | Удалить установленный голос           |
| GET    | `phrases`                      | Список ранее сгенерированных фраз     |
| POST   | `phrases`                      | Сгенерировать фразу из текста         |
| GET    | `phrases/{id}:download`        | Скачать сгенерированный `.wav`        |
| DELETE | `phrases/{id}`                 | Удалить запись из истории             |

## Дорожная карта

- **Фаза 1 (этот релиз)**: отдельная страница студии + REST API + скачивание
- **Фаза 2**: встраивание кнопки «Сгенерировать через TTS» в существующую страницу
  редактирования звуковых файлов (рядом с записью с микрофона и загрузкой файла)
